Panduan BlokBuild Pro

Semua yang perlu Anda tahu untuk membuat website profesional tanpa coding. Ikuti dari atas, atau lompat ke topik yang Anda butuhkan lewat menu kiri.

🚀 Mulai Cepat

Membuat website pertama Anda hanya butuh 3 menit:

1
Buka aplikasiKlik dua kali file BlokBuild-Pro.html — akan terbuka di browser (Chrome, Edge, atau Firefox).
2
Masukkan kunci lisensiKetik kunci yang Anda terima, lalu klik Aktifkan.
3
Pilih cara mulaiKlik ✨ AI Generate dan ketik ide Anda, atau 📑 Template untuk desain siap pakai, atau tarik blok manual dari kiri.
4
Edit & downloadKlik blok untuk mengubah teks/warna, lalu klik ⬇️ Download untuk dapat file HTML siap pakai.
💡 Tips: Tidak perlu internet untuk memakai aplikasi. Semua berjalan di browser Anda.

🔑 Aktivasi Lisensi

Saat pertama membuka aplikasi, akan muncul layar kunci. Masukkan kunci lisensi Anda dan klik Aktifkan. Kunci hanya perlu dimasukkan sekali per browser — selanjutnya aplikasi langsung terbuka.

⚠️ Jika kunci ditolak, pastikan tidak ada spasi di awal/akhir dan huruf ditulis kapital. Hubungi penjual jika masih bermasalah.

✨ AI Generate

Fitur andalan BlokBuild. Cukup tuliskan ide website Anda dalam bahasa biasa, AI akan menyusun halaman lengkap otomatis — termasuk warna, struktur, dan konten awal.

1
Klik tombol ✨ AI Generate
2
Tulis ide AndaContoh: "buat landing page untuk cafe Kopi Senja yang jual kopi premium dan suasana cozy"
3
Klik Buat SekarangAI menyusun hero, fitur, galeri, harga, FAQ, form, dan footer sesuai jenis usaha Anda.

AI mengenali berbagai jenis usaha (cafe, toko, jasa, klinik, fashion, properti, edukasi) dan menyesuaikan tema warna otomatis. Setelah dibuat, Anda tetap bisa edit setiap blok.

💡 Semakin spesifik ide Anda, semakin tepat hasilnya. Sebutkan jenis usaha + nama + ciri khasnya.

🧱 Bekerja dengan Blok

Website dibangun dari "blok" — potongan siap pakai seperti Hero, Galeri, atau Form. Ada 26 jenis blok dalam 5 kategori.

Menambah blok

Tarik blok dari panel kiri ke kanvas tengah. Posisi sisip ditentukan oleh tempat Anda melepas.

Mengatur blok

Klik blok untuk memilihnya, lalu gunakan tombol di pojok kanan blok:

✏️ Edit Properti

Setelah memilih blok, panel kanan menampilkan semua properti yang bisa diubah: teks, warna, ukuran, perataan, dan link. Perubahan langsung terlihat di kanvas (WYSIWYG).

Untuk mengganti warna, klik kotak warna atau ketik kode hex langsung (misal #d4622a).

⚙️ Mode Advanced

Untuk pengguna teknis. Di panel kanan, tab Advanced memungkinkan Anda menambahkan:

  • ID Elemen — untuk anchor link atau target JavaScript
  • Class CSS — untuk styling lanjutan
  • Inline Style — CSS tambahan langsung pada blok (misal margin-top:40px; box-shadow:0 4px 12px #0002;)
💡 Pengguna awam bisa abaikan tab ini sepenuhnya — aplikasi tetap berfungsi penuh tanpanya.

📄 Multi-Halaman

Buat website dengan banyak halaman (Beranda, Tentang, Kontak, dll). Tab halaman ada di atas kanvas.

  • Tambah halaman: klik tombol ＋ di bar tab
  • Ganti nama: klik dua kali pada tab halaman
  • Hapus: klik tanda × pada tab

Saat di-export, setiap halaman jadi file HTML terpisah dengan menu navigasi otomatis yang menghubungkan semua halaman. Halaman pertama selalu menjadi index.html.

🔍 SEO & Meta

Agar website mudah ditemukan di Google dan tampil bagus saat dibagikan. Buka ⚙️ Pengaturan dan isi bagian SEO:

  • Deskripsi — ringkasan website (ideal 150-160 karakter)
  • Kata kunci — pisahkan dengan koma
  • Gambar Share (OG Image) — gambar yang muncul saat link dibagikan di WhatsApp/Facebook (ukuran ideal 1200×630)
  • Favicon — ikon kecil di tab browser
  • Google Analytics ID — untuk melacak pengunjung (format G-XXXXXXXXXX)

🎨 Tema & Efek

Di ⚙️ Pengaturan bagian Tampilan:

  • Tema Warna — 6 pilihan (Oranye, Biru Laut, Hijau Hutan, Ungu, Pink, Hitam Putih). Mengubah seluruh skema warna sekaligus.
  • Animasi scroll — blok muncul halus saat di-scroll
  • Tombol mode gelap — pengunjung bisa beralih ke tampilan gelap

🗄️ Form ke Database (Supabase)

Blok "Form → Supabase" menyimpan data pengunjung langsung ke database gratis Supabase.

1
Buat akun & project di SupabaseGratis di supabase.com
2
Buat tabelDengan kolom sesuai nama field di form Anda (misal: nama, email, no_hp)
3
Aktifkan RLS Policy INSERTAgar pengunjung bisa mengirim data (untuk anon)
4
Isi koneksi di PengaturanSupabase URL, Anon Key, dan nama tabel
⚠️ Anon Key aman dipakai di website publik karena hanya berfungsi sesuai aturan RLS yang Anda tetapkan. Jangan gunakan service key.

🛒 Order via WhatsApp

Dua blok khusus untuk usaha:

  • Order via WhatsApp — pengunjung pilih produk + jumlah, klik, pesanan langsung masuk ke WhatsApp Anda lengkap dengan total harga.
  • Booking / Reservasi — form reservasi (nama, layanan, tanggal, jam) yang terkirim ke WhatsApp.

Cukup isi nomor WhatsApp Anda dengan format 62 (misal 628123456789) dan daftar produk dengan format Nama|Harga.

🌐 Hosting Website (Gratis)

Setelah download, website Anda perlu di-hosting agar bisa diakses publik. Pilihan gratis termudah:

Cara tercepat: Netlify Drop

  1. Buka app.netlify.com/drop
  2. Seret file HTML (atau isi folder ZIP yang sudah diekstrak) ke halaman tersebut
  3. Selesai — Anda langsung dapat link website yang bisa dibagikan

Alternatif: GitHub Pages

Upload file ke repository GitHub, aktifkan Pages di Settings. Cocok untuk yang sudah punya akun GitHub.

💡 Untuk domain sendiri (misal tokosaya.com), beli domain lalu hubungkan ke Netlify/GitHub Pages melalui pengaturan DNS.

❓ Masalah Umum

Tombol/perubahan tidak muncul?

Refresh browser (Ctrl+Shift+R) dan pastikan memakai file versi terbaru.

Form Supabase tidak mengirim?

Cek URL & Anon Key benar, nama tabel & kolom sesuai, dan RLS Policy INSERT sudah aktif.

Project hilang setelah tutup browser?

Project tersimpan di browser yang sama. Untuk backup permanen atau pindah komputer, gunakan Export .json.

Peta/video tidak muncul?

Kedua fitur ini butuh koneksi internet saat website dibuka pengunjung.
